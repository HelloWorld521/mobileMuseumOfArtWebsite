# mobileMuseumOfArtWebsite

The Art Exchange platform between students and enterprises   
<br>
**系统介绍:**   
<br>
该系统主要用来给学生提供一个与企业交易的一个平台。企业可注册网站，将自己的设计需求发布到网站上，学生搜索自己感兴趣的设计，向企业投稿，企业选择投稿进行交易。学生还可以在自己的个人界面上传自己的作品，展示出来。
